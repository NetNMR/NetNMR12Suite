The Output Window contains three tabs for the peak-picking, integrals and the shelf. One of these three tabs will be display automatically when clicking on the Peaks, Integral and Shelf Ribbon tabs.

Each output tab will display a header line to indicate the values of each item in each subsequent line. While this header line can be seleted, its selection has no effect on any subsequent command in that window. Once a peak or integral is selected in either of the output windows, these items will show up in a highighted color on the display. The color can be adjusted by the user in the Properties Window.

Each output tab has a specifc context menu:

The Peaks output window contains a menu command for setting the offset of a selected peak. If more than one peak is selected, this command will be disabled. Selected peaks, or the entire Peaks tab can be copied to the clipboard with other commands in the context menu. The header line will always also be copied to the clibpoard.

The Integral output window contains a context menu command for setting the integrtion value. Either a single integral can be selected or multiple integrals selected. If multiple integrals are selected, the sum of the selected integrals will be used to set the integral constant. Context commands can also copy or delete selected integrals or all integrals.

The Shelf output window contains context menu command for copying the shelf to the clipboard and for deleting either selected shelf regions or the entire shelf.
