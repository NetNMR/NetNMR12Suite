Colors for the spectrum display are selected in the "Display Colors" branch of Properties Window.

One of two basic color schemes can be selected with a Color Scheme drop-down selection: "Default" is a colorful display scheme using a black background and "Black on White" is as its name suggests and is more suitable for copying graphics to another application, like PowerPoint™.

Individual colors can be set for a variety of graphic elements in the Net12NMR display in the "Item Colors" branch. Just click on a graphic item name and then select the drop-down control to assign a different color. A standard color can be choosen or any custom color can be selected from here. Once a color has be customized, the Color Scheme drop-down selection will show "Custom".
