The Header ribbon tab is used to manange the flexible header of the current active NMR file. The flexible header is esentially a string-based name-value dictionary. Select a flexible header item name from the "Name" pane and the value will be display in the "Value" pane. The value can be changed by editing the string in the value edit box and pressing the Update command button. You can delete a flex header name-value by selecting the name and pressing the Delete button. You can also create a new name-value entry in the flex header by typing the new name in the "New Item" pane and then pressing the Createbutton.
