The Quick Access Toolbar at the top of the Net12NMR main window is a place where frequently used commands are always available. It can be customized by the user by clicking on over-scored down arrow just to the right of the last command and selecting More Commands....

In the Customize dialog, select a set of command by selecting an area from the "Chose commands from:" drop-down selection. Only choose commands that are represented by a small black background icon. These are the only commands that can be executed at any time. Many of the commands on the various tabs of the ribbon bar are only appicable while that ribbon bar tab is displayed. Most of the commands which may be useful to install on the quick access toobar are in the Home Tab, and the File menu.
