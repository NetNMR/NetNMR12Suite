Choose Color dialog box

The Choose Color Dialog is used to select all the colors used in the display window. Use the drop-down to select a graphic element of the display. The current color will be displayed in a small rectangle just to the right of of the drop-down. To change the color, click the Change... button to bring up a the standard dialog to select a new color.

Buttons are also available to set the colors to three different standard color schemes: black on white, white on black, and the standard color scheme.
