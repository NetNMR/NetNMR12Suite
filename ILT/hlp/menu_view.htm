View menu commands

The View menu offers the following commands:

Next Shows the next decay dataset.
Previous Shows the previous decay dataset.
Decay Shows a dialog uses to select the a specific decay from a multiple decay dataset.
Text Shows a dialog containing a text readout of the current decay, including any fit.
Confidence The Confidence menu has a popup submenu for selecting confidence intervals to be displayed for the density function plot.
Transform This toggle selects between the observed data and the fit density function. This toggle is not active if a fit has not yet been generated.
Toolbar Shows or hides the toolbar.
Status Bar Shows or hides the status bar.
