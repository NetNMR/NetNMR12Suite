File Open dialog box

The following options allow you to specify which file to open:

File Name

Specifies the file you want to open. This box lists files with the extension you select in the List Files of Type field.

List Files of Type

Specifies the type of file you want to open:

*.ILT files are native files created by this application. These files will include the observed data, made up of one vector of X-values and at least one vector of observed Y-values. Terms used to fit each observed Array1D are also included.

*.TXT files can also be used to input the X-value vector and multiple observe vectors. See Input Text File Format for more information.

Drives

Specifies the drive in which ILT stores the file that you want to open.

Directories

Specifies the directory in which ILT stores the file that you want to open.

Network

Click this button to connect to a network location, assigning it a new drive letter.
