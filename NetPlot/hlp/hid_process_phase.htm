Phase Dialog

The Phase Dialog is used to phase recently Fourier transformed spectra. The button commands on this dialog only work on data that have been added during the current session and only if the raw FID files have been imported and then transformed using the Process dialog. Please note that imaginary data will not be saved in a Netplot (*.npl) file, so data must be phased during the same session that the file data was added. Use the "Applies to" feature to direct the current command to either an individual spectrum, or all the spectra in the current NetPlot document. After phasing, any baseline correction previously done on the spectrum will be lost. This dialog contains the following buttons:

All checkbox This checkbox determine the object of each of the phase commands. Checking this box will apply the desired phase correction on all of the data in the current document. Unchecking this box will activate the ajacent dropdown and a single spectrum in the current NetPlot document can be selected.
All Others button This command is only available if the All checkbox is currently not selected. This command will propagate the current global phase correction to all spectra in the current NetPlot document except the spectrum currently selected in the Apply to: dropdown.
Up button This command is only available if the All checkbox is currently not selected. This command will propagate the current global phase correction to all spectra above the spectrum currently selected in the Apply to: dropdown.
Down button This command is only available if the All checkbox is currently not selected. This command will propagate the current global phase correction to all spectra below the spectrum currently selected in the Apply to: dropdown.
Auto A Only button The linear global phase correction is set to zero and the global phase constant is varied for the spectrum to automatically optimze the phase. The spectrum range of this command is determined by the controls in the Apply to: selection box.
Auto A & B button Both the linear global phase correction and the global phase constant is varied for the spectrum to automatically optimze the phase. The spectrum range of this command is determined by the controls in the Apply to: selection box.
Magnitude button The phase-independent magnitude spectrum is calculated.
At Cursor button This command is only enabled when the All checkbox is unselected. When this button is pressed, the NetPlot display only displays the selected spectrum, allowing the user to mouse-adjust the phase at the position of the two zoom cursors. Keyboard commands L and R will move between the left and right zoom cursors, respectively. (Alternately, the left mouse button can be used to toggle between the left and right cursor phase modes.) the phase of the spectrum of the other cursor will be held in place while the user move the mouse up and down. (Right and left mouse motion will be ignored.) The Shift and Control keys can be used to respectively amplify and soften the effect of the mouse movements on the computed phase correction. When the user is satisfied with the current phase correction, a C key press will terminate the At Cursor phase mode. While in this interactive phasing mode, the vertical expansion of the spectrum can be adjusted up or down with the U and D keys. The current phase can be reset with a T key press. Alternatively, you can immediately move to the next spectrum (above or below) by pressing the Q or Z keys. In this case the phase at the current spectrum will then define the global phase spectrum and this will be applied to the next spectrum. The current expansion and zoom regions as well as the location of the zoom cursors will be maintained, making it easy to phase an entire NetPlot Document. The L, R, U, D, T, C, Q and Z keyboard command are also available from a context menu by clicking the right mouse button while in the At Cursor phase mode.
