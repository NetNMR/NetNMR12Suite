This command displays a dialog that will show all spectrum text labels. (A label value must first be specified in the Parameter dialog.) Each spectrum label is listed on a single line, with the same top-to-bottom order as the spectra. You can perform any sort of editing you like. Blank lines will effectively remove a spectrum label. If you add additional lines, the extra lines at the end of the dialog will not be used when the spectrum display is updated. If you remove lines in this dialog, this will effectively remove labels from spectra at the bottom of the display. The label font size can also be specified.

Changing a label value will not alter the spectrum header value. If you need to reinitialize the display with the values of a particular variable, you must do this in two steps. First, use the Parameter dialog to change (or clear) the name of the label parameter and press “OK”. Then use the Parameter dialog and change the name of the label parameter back to the original value.

The accelerator for this command is Control+L.
