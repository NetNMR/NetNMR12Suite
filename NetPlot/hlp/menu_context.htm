The context menu is accessed by right-clicking on the display. The Phase dialog has its own, unique context menu. The menu currently has the following commands:

Zoom Changes the display range to be where the zoom cursors are currently located.
Full Changes the display range to the maximum required to view all spectra.
Y-scale The Y-scale submenu has commands Up, Down and X1 to control the Y Expansion Factor.
Set Offset Here The user will be prompted for a new chemical shift value at the current cursor position. The units (PPM or Hertz) will be determined by the current Single Point Region Adds a single point at the cursor position as a new region.
Add Region Adds the current region between the zoom cursors to the region(s).
Delete Region Removes region between the zoom cursors from the current region between the zoom cursors.
Clear All Removes all regions from the current document.
Fit Integrals Integrals regions in the current display are calculated and sent to a spreadsheet for analysis.

The availablity of these command will depend on the current state of the display.
