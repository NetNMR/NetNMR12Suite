This command will "Unzoom" the display range to the full region. This new region will be wide enough to display all points in all of the spectra and the display will include a chemical shift axis. This will not set the view mode to None but will keep the view mode in either PPM or Hertz. If the current mode set in the Parameter dialog is None when this command is executed, the display mode will be set to PPM.

This command is available both on the standard View Menu and the Context Menu.
