Accelerators a keyboard operations to execute a menu command. The NetPlot accelerators are:

Accelerator Command
Shift+F1 Context Help
Control+Shift+C Edit->Colors
Control+C Edit->Copy
Control+Insert Edit->Copy
Control+L Edit->Labels
Control+P Edit->Parameters
Control+A File->Add
Control+N File->New
Control+O File->Open
Shift+P File->Print
Control+S File->Save
Up Arrow View->Y-scale->Up
Down Arrow View->Y-scale->Down
Home View->Y-scale->x1
