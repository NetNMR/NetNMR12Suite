Single Point Region Command (Context Menu)

A single point at the main cursor is added to the regions.

This command will only be active when the Add Region command is active.

Please note that if you mix and match broad integral regions with single point integrals, subsequent data fitting in the resulting speadsheet built with the Fit Integrals command may be tricky because the integrals from a single point can be very different from a broad integral region. This can make the "solver" spreadsheet function work harder. In some cases individual curve should be solved one at a time.
