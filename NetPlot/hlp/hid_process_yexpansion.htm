Y-Expansion Dialog

The Y-Expansion Dialog is used to adjust the y-scale of individual spectra in the NetPlot document. Adjusting the y-expansion of individual spectra does not affect any integral of that spectrum, just like the Global scaling control. The user selects a spectrum and then adjust the yscale of that spectrum. The scale factor can be explicitly set through a text-edit control or can be adjusted with both the mouse wheel or the keyboard UP- and DOWN-Arrow. The arrows change the scaling by a factor of 2 while each click of the mouse wheel move the scale the fourth root of two, or about 19%. If the Control key is held down while using the wheel, the effect will be smaller, the sixteenth route of two, about 4%. This dialog contains the following controls:

Spectrum Drop-down Selection This drop-down selection tool is used to select the individual spectrum to be adjusted.
Change Button This button is used to load the y-scale parameter from the ajacent text edit control.
Edit Control The new y-scale value is specified here. Once the value has been input, press the Change button next to this edit control. Valid values are any real positive number. A number greater than 1.0 will cause the slected spectrum to grow while a value less than 1.0 will cause the selected spectrum to shrink.
Reset Button This button will reset all individual y-scale factors to 1.0.
Global Checkbox Not actually part of the individual y-scale control, but there for continuity, this checkbox will turn on and off global scaling.
