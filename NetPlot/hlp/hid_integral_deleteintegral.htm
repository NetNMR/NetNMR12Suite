Delete Region Command (Integral Menu)

This will remove the current region between the zoom cursors.

This command will only be active if the zoom cursors are visible and there is at least one integral region between the zoom cursors and the current display mode is not None and each of the spectra in the current document were obtained with identical spectrum windows. That is the sweep width, spectrometer frequency, offset and number of points must be the same for each and every spectrum in the current NetPlot document.
