File Save As dialog box

The following options allow you to specify the name and location of the file you're about to save:

File Name

Specifies a file name to save a document with a different name. NetPlot adds the extension you specify in the Save File As Type box.

*.npl stores the file in a native format. If all of the spectra in the current document have the same spectrum width, offset, spectrometer frequency and number of points, the *.txt file type will also be available. Tyis type will output the spectra in a text-based matrix. The first column will be the Hertz or PPM of each point in that row of data. The top row will be the label of each spectrum and all other cells will contain the intensity of the spectrum at that Hertz or PPM.

Drives

Specifies the drive in which you want to store the document.

Directories

Specifies the directory in which you want to store the document.

Network

Click this button to connect to a network location, assigning it a new drive letter.
