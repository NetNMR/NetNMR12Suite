The NetPlot display mode is set primarily with the Parameters dialog. There are three display modes:

None No axis is drawn in this mode. Each spectrum is exactly the same width on the plot and there is no offset according to the offset of each individual spectrum.
PPM Includes an axis in PPM. Each spectrum is drawn in relaxation to the axis.
Hertz Includes an axis in Hertz. Each spectrum is drawn in relation to the axis.

Once the display mode is set to either PPM or Hertz, a main cursor will track the mouse arrow. The main cursor is a vertical line drawn over all the spectra in the current document. The chemical shift of the main cursor position is listed in the status panel at the bottom of the window. In addition, the zoom cursors can then be set by click-drag-release action with the left mouse button.

If all the spectra in the current document have identical sweep widths, spectrometer frequencies, offsets and number of points, regions can be selected and baselines corrected and integrals calculated over any number of regions with the Regions dialog, but only if the display mode is not set to None.

If the Full menu command (available in both the View Menu and the Context Menu) is executed in the None display mode, it will be changed to PPM automatically.

There are special display modes associated with phasing and baseline correction. When using the At Cursor phasing command, the display only shows a single spectrum from the NetPlot document while the user adjusts the phase. In this mode, the cursor is hidden from the user and a special context menu is available for manual phase. For more information, see the How To... manually phase an entire document.

After pressing the Baseline: Calculate button, the NetPlot will display calculated polynomial baseline corrections along with the spectra in the current NetPlot document.

Another special display mode is when ajusting indivial spectrum Y-scale factors with the Y-Expansion dialog. Anytime this dialog is displayed the UP-Arrow, DOWN-Arrow and the mouse wheel control the individual, selected spectrum Y-scale. Whenever this dialog is not display, these controls adjust the Y-scale of all the spectra.
