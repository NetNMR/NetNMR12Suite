While phasing command buttons on the phase dialog can be performed on all spectra in the current NetPlot document, this is not desirable in some cases. Often, various kinds of relaxation experiments will display a small phase shift from spectrum to spectrum and autophase my not produce an acceptible phase for each spectrum. In that case the At Cursor button command is the preferred technique for phasing.

It is not necessary to perform inital automatic phase corrections on each of the spectra, because the At Cursor phase mode will automatically update each spectrum with the current global phase when moving between the different spectra.

Start manually phasing by first selecting either the top-most or bottom-most spectrum in the Applys to: selection drop-down list. (The All checkbox must cleared before an individual spectrum can be selected. Before pressing the At Cursor button, the zoom cursors should be placed on resonances near the left and right side of the current displayed region. (If the At Cursor button is pressed before the zoom cursors are set, NetPlot will remind you to set the zoom cursors.)

After pressing the At Cursor button, the selected spectrum will be displayed in the middle of the display window, without the boundry sheet. The mouse cursor will disappear and vertical mouse motion will move the phase of the spectrum around the left or right cursor. (the current location of the control point will be display by a single zoom cursor.) As the mouse is moved up and down the phase of the entire spectrum will be adjusted while holding the non-selected zoom cursor point at zero. You can select between the left and right zoom cursors by either typing L or R on the keyboard or pressing the left mouse button.

It is generally useful to highly expand the vertical scale of the spectrum so that the phasing around the base of the peaks can be seen in detail. The vertical expansion can be adjusted up or down with the U or D keys, respectively. Iterating between the left and right zoom cursors a couple of times is usualy sufficient to produce a satisfactory phase correction.

While you can apply this current phase correction to all other spectra in the current document by typing a C to exit the manual phase mode and then pressing one of the buttons in the Clone Phase: section, you can switch to the next spectrum without leaving the manual phase mode. The Q and Z keys move to the spectrum above or below the currently selected spectrum. Moving between spectra in this way perserves the current Y-scale and immediately applies the current global phase to the new spectrum. In this way you can quickly phase an entire NetPlot document. If you find that spectra are immediately acceptable without further mouse adjustment and it is susupected that all of the remaining spectra will not need any special adjustment, you can exit the manually phasing mode by typing C and then using one of the Clone Phase: buttons to propagate the current global phase to the remaining spectra.

If you want to move to a spectrum that is not adjcent to the current spectrum and don't want to affect the phase of the spectra between, exit the manual phasing mode by typing C and then selecting the desired spectrum in the Apply To: drop-down selection list and finally pressing the At Cursor button.

When in the At Cursor manual phase mode, there are several special keyboard commands available:

L Move to the left cursor.
R Move to the right cursor.
C Exit (complete) the manual phasing mode.
T Reset the phase to the last global phase point. (The last global phase point is saved when moving between spectra with the Q or Z keys.)
U Blow up the vertical expansion.
D Shrink down the vertical expansion.
Q Move to the spectrum above. If you are already at the top of the NetPlot document, this command will do nothing.
Z Move to the spectrum below. If you are already at the bottomof the NetPlot document, this command will do nothing.
