Regions Dialog

The Regions Dialog is used to select regions of the current NetPlot document. The current region between the zoom cursors can be added or removed with command buttons on this dialog. Up to a 10-term simple polynomial can be fit to the spectrum using all the spectral point not currently in a region. In addition, integrals of the current selected regions can be analyzed with an accompanying spreadsheet macro. In usual practice, the baseline is corrected first by selecting any region that needs to be exculded from the baseline, i.e., contains no peak intensity. Then the regions can be optionally modified and then passed to an external program for analysis. This dialog contains the following buttons:

Add button The region currently between the zoom cursors will be added to the regions. This region will be display in an alternate, user-selectable color.
Clear button The region currently between the zoom cursors will be removed from the regions. This region will be display in an default, user-selectable color.
Clear All All seleceted regions will be removed from the current NetPlot document.
Order dropdown The polynomial order of the baseline correction is slected. Valid values range from 2 to 10.
Calculate pushbutton The polynomial correction for each spectrum is calculated and displayed in a user-selected color. In a pused state, this button is relabeled "Apply". To apply the current selection, push this button again.
Cancel button This will exit the current baseline calculation and display with applying any correction.
Excel® button The integral of each region of each spectrum in the current NetPlot document is calculated and copied to a new Excel® workbook. Add-in commands are then available to build an analysis spreadsheet based on a user-selected function. Excel's solver can then be used to fit the function parameters to the integral data.
ILT button The integral of each region of each spectrum in the current NetPlot document is calculated and passed to ILT.exe for analysis by inverse LaPlace transform. ILT.exe can fit the data to a distribution of exponential functions.
