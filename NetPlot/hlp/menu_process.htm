Process menu commands

The Process menu offers the following commands:

Process Displays a Process dialog with various buttons to perform standard process routines to raw (FID) data.
Phase Displays a Phase dialog with various controls for adjusting the phase of each spectrum.
Subtract Displays a Subtract dialog for subtracting a spectrum, or a spectrum average from all other spectra in the document.
Regions Displays a Regions dialog for adjusting the selected regions of all the spectra for subsequent baseline corrections and/or integrations.
Y-expansion Displays a Y-expansion dialog for adjusting the Y-expansion of individual spectra.
