Edit menu commands

The Edit menu offers the following commands:

Copy Copies data from the document to the clipboard.
Spec Frequency Used to change the spectrometer frequency for calculating line widths in Hertz.
Spectra Dialog used for managing spectra in the current document.
Lines Dialog used for managing lines in the current document.
Fit Used to optimize the line parameters (height, shape, width and shift) to fit the spectra in the current document
