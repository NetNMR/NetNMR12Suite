Zoom command (Context menu)

Zoom cursors must first be set across a region of the spectrum. Do this by clicking with the left mouse button on one side of the desired zoom region and while holding the left mouse button down, sweep across the desired zoom region, releasing the left button on the other side of the desired zoom region. This action will place two vertical zoom cursors on the region. Then, anywhere on the spectrum, right click and select zoom. The zoom command will be disabled whenever there are no zoom cursors, or if there is a line in the current NetSpan document that lies outside the zoom cursors.
