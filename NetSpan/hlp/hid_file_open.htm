Open command (File menu)

Use this command to open an existing document in a new window. Use the Window menu to switch among the multiple open documents. See Window 1, 2, ... command.

You can create new documents from a previously saved NetPlot document (*.npl). Because of existing NetSpan requirements, after the first spectrum is imported from a NetPlot document, only the remaining spectra in the NetPlot document with an identical spectrometer frequency, sweep-width, offset and size will be imported.

You can also import text (*.txt) documents. These are most easily created by NetPlot. If you are interested in this feature, use NetPlot to save a text document. The format of this document is easily determined by simple examination.

You can create new documents with the New command.
