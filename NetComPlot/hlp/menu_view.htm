View menu commands

The View menu offers the following commands:

Information Toggles the display of the 2D file names at the top of the page.
Portrait The page is set to portrait orientation.
Landscape The page is set to landscape orientation.
Zoom Use the mouse pointer to drag a box around a region of the 2-D spectrum and you can quickly zoom to that region with this menu command.
Full Return to displaying the full width of all 1- and 2-D spectra in a composite plot.
Previous Return to displaying the previous display range.
Integral Calculates and displays integral(s) of each of the 2D spectra inside the zoom box.
Toolbar Shows or hides the toolbar.
Status Bar Shows or hides the status bar.
