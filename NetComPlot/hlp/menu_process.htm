Process menu commands

The Process menu offers the following commands used to symmetrize the top level 2D data. These commands will only be active for square 2D data that has identical spectral windows.

Symmetrize(Min) Points across the diagonal are replaced with the minimum of the two symmetry related points. If one point is negative and one is positive amplitude, both are set to zero.
Symmetrize(Ave) Points across the diagonal are replaced with the average of the two points.
