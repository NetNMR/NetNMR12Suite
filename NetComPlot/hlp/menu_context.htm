Context menu commands

The Context menu is accessed by right-clicking on the main window:

Zoom Use the mouse pointer to drag a box around a region of the 2-D spectrum and you can quickly zoom to that region with this menu command.
Full Return to displaying the full width of all 1- and 2-D spectra in a composite plot.
Set Offset Here You can change the offset of a 1- or 2-D spectrum with this menu command.
